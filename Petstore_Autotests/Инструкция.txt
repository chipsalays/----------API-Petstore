Для запуска автотестов необходимо установить Python версии 3.8 или выше с официального сайта. При установке обязательно поставить галочку Add Python to PATH. 
После установки открыть командную строку через Win+R, ввести cmd и нажать OK.
 В командной строке выполнить команды для установки библиотек: pip install requests и pip install pytest.
 Далее скачать с GitHub репозиторий Petstore_Autotests или создать на рабочем столе папку с названием Petstore_Autotests и поместить в неё файл test_get_user.py.
 После этого в командной строке нужно перейти в папку с тестами. Если папка находится на рабочем столе, команда будет: cd C:\Users\%USERNAME%\Desktop\Petstore_Autotests.
 Если папка находится в другом месте, нужно заменить путь на свой.
 Затем запустить тесты командой pytest -v -s. Результат покажет 7 тестов: тесты 1, 2, 3, 4 и 6 должны быть PASSED, тесты 5 и 7 будут FAILED. Это не ошибка кода, а обнаруженные баги сервера Petstore. 
Для сохранения результата в файл использовать команду pytest -v -s > result.txt. Файл result.txt появится в папке с тестами.
